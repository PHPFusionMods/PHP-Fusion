--------------------------------------------------------------
Norsk språkpakke til PHP-Fusion versjon 6.00.400
--------------------------------------------------------------

Les dette:

Denne oversettelsen skal bare brukes med versjon 6.00.400.


---- Installere: ---------------------------------------------

>	Pakk ut (unzip) mappen "norwegianlocale6-00-4.zip"

>>	Last opp mappen "Norwegian" til mappen "Locale" 
	på din php-fusioninstallasjon via FTP

>>>	I hovedinstillinger (Main Settings) på siden din, velg
	språk (language) Norwegian. Lagre.
